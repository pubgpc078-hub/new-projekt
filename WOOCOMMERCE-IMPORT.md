# راهنمای درون‌ریزی محصولات در ووکامرس

فایل `woocommerce-products.csv` شامل ۲۴ محصول فروشگاه «منوجان کالا» با فرمت استاندارد ووکامرس است و **بدون نیاز به افزونه** قابل وارد کردن است.

## مراحل درون‌ریزی

1. وارد پیشخوان وردپرس شوید.
2. به مسیر **محصولات → همه محصولات** بروید و روی دکمه‌ی **درون‌ریزی (Import)** بالای صفحه کلیک کنید.
3. روی **Choose File** بزنید و فایل `woocommerce-products.csv` را انتخاب کنید → **Continue**.
4. در صفحه‌ی نگاشت ستون‌ها (Column mapping)، ووکامرس ستون‌ها را به‌صورت خودکار تشخیص می‌دهد. فقط بررسی کنید همه‌چیز درست نگاشت شده باشد → **Run the importer**.
5. منتظر بمانید تا درون‌ریزی کامل شود.

## نکات مهم

- **واحد پول:** قیمت‌ها به **تومان** هستند. از مسیر **ووکامرس → تنظیمات → عمومی**، واحد پول را روی تومان/ریال ایران تنظیم کنید تا اعداد درست نمایش داده شوند.
- **تصاویر:** آدرس عکس‌ها از Unsplash است. هنگام درون‌ریزی، ووکامرس آن‌ها را دانلود و در رسانه‌ها ذخیره می‌کند؛ پس باید سرور شما به اینترنت دسترسی داشته باشد و این مرحله ممکن است کمی طول بکشد.
- **قیمت حراج:** محصولات تخفیف‌دار به‌صورت خودکار «قیمت اصلی» و «قیمت حراج» دارند و نشان تخفیف نمایش داده می‌شود.
- **ویژگی‌ها:** «برند» و «گارانتی» به‌صورت ویژگی (Attribute) سراسری اضافه شده‌اند تا برای فیلتر کردن محصولات استفاده شوند.
- **دسته‌بندی‌ها:** ۶ دسته (یخچال و فریزر، ماشین لباسشویی، کولر گازی، جاروبرقی، لوازم آشپزخانه، تلویزیون) به‌صورت خودکار ساخته می‌شوند.

## ساخت مجدد فایل

برای تولید دوباره‌ی فایل (مثلاً پس از تغییر محصولات):

```bash
python3 scripts/generate_woocommerce_csv.py
```
